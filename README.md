## üöÄ Getting Started

### Prerequisites

* Node.js 18.17 or later
* npm, yarn, or pnpm

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/political-test.git
   cd political-test
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the development server:

   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.
